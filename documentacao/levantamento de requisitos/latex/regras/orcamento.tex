\chapter{orçamento}
Para construir ou reformar é preciso conhecer as etapas de uma obra, desde a contratação dos projetos de arquitetura até a limpeza do local.

O orçamento de obra envolve as etapas de elaboração de um projeto \footnote{obra de construção civil, projeto estrutural, etc.} de construção determinando detalhadamente o custo de sua aplicação, especificando as atividades necessárias para a aplicação do projeto (comumente nomeadas de \emph{serviços}), se aprofundando nos custos envolvidos para a execução de cada atividade, desde mão-de-obra e custo de material básico como cimento e areia, da utilização de recursos externos tais como equipamentos alugados e até mesmo mão-de-obra especializada, impostos envolvidos nas atividades, entre outros.

Existem 3 fases diferentes de orçamento:

\begin{itemize}
	\item Orçamento de estimativa;
	\item Orçamento de custo real; e
	\item Orçamento de execução.
\end{itemize}

\section{Orçamento de estimativa}
Esta é a fase inicial do orçamento, quando o cliente solicita um orçamento prévio sobre um determinado serviço que deseja realizar, sendo esse somente uma estimativa pois não neste momento não há um projeto definido, não se conhecem todas as atividades necessárias para sua execução.

Normalmente na fase de estimativa levam-se em conta projetos base \footnote{projetos criados para servir de base para outros, tendo pequenas diferenças em suas atividades} para facilitar a recuperação de atividades padrão, sendo modificadas somente algumas atividades em particular ao projeto que se estima. Além de utilizar projetos base, mais comumente são utilizados padrões de projetos (apostilas que determinam os elementos base de um determinado projeto). Nesses casos, o usuário que define o projeto deverá seguir os padrões pré-estabelecidos para manter a conformidade de suas atividades.

\section{Orçamento de custo real}
Nesta fase, o orçamento é baseado em projeto bem definido pelo engenheiro, tendo suas atividades bem definidas, quantidade de recursos humanos e materiais definida, prazos e duração das atividades estabelecidos, impostos determinados, entre outros.
A obtenção do custo real é baseada na estruturação dos serviços e suas composições, determinadas pela quantidade de cada item para a execução do serviço, como mostra a figura \ref{img:composicao}.

A somatória dos serviços e recursos necessários, juntamente com a aplicação dos insumos e impostos referentes representam o custo real de execução do projeto.

\section{Orçamento de execução}
O orçamento de execução é a fase final do orçamento, este considerando todos os custos de serviços de execução, custos indiretos (estimados pois os custos indiretos reais somente serão identificados ao final do cronograma de execução), taxas (insumos e impostos) referentes às atividades a serem executadas e o BDI\footnote{Bonificação e Despesas Indiretas - também conhecido como LDI(lucro e despesas indiretas) - é aonde entra o lucro da obra.}.

Em suma, o orçamento de execução é a fase em que o mesmo é realizado para aprovação de execução pelo cliente.
